About Kado Permissions Tracker

Ever wondered what changed on the default roles you were using for your workflows? Use Kado Permissions Tracker to stay up to date with the latest changes.

This tool is part of the Kado ecosystem, created by Janpreet Singh (janpreet.com). Reach out if you have any questions, and be sure to read the README on the GitHub repository for more information.
